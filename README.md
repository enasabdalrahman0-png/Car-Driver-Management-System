#  Car Driver Management System

A Java-based application for managing car and driver information.

##  Features
- Add drivers
- Assign drivers to cars
- Display driver details
- Simple data management system

##  Technologies Used
- Java

##  Description
This project started as a lab exercise and was enhanced into a functional system for managing car and driver data using Java.

##  How to Run
1. Open the project in any Java IDE
2. Run the main file
